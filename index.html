Sir Accordion

To expand a content given a coord e.g: 1-1, put the coord in the input and click the expand buttom


Awesome dynamic, recursive, customizable and multilevel Accordion Menu for AngurlarJS, that builds up from a JSON object and an optional configuration object. You can download the project for a live demo in the example folder (don't forget to bower install dependencies) and for better understanding.

No jQuery needed

You could set before/after topContent/bottomContent in the accordion config object to set a scroll back to top buttom (or whatever you like), given that you only need to set the value once and it will be in all the contents of the accordion.

Go to the Project page for a how to use
